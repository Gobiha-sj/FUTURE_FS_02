# CRM System – MERN Stack

A full‑stack **Customer Relationship Management (CRM)** application built with **React (frontend)**, **Node.js/Express (backend)**, and **MongoDB Atlas (database)**.  
Deployed on **Netlify (frontend)** and **Render (backend)**.

---

## 🚀 Live Demo
- **Frontend (React):** [gobihasj-crm.netlify.app](https://gobihasj-crm.netlify.app)  
- **Backend (API):** [https://crm-fs-l1mw.onrender.com](https://crm-fs-l1mw.onrender.com)

## ✨ Features
- 🔐 User authentication (register/login with JWT)  
- 📋 Manage leads (add, view, delete)  
- 🖥️ Responsive dashboard UI  
- 🌐 Cloud‑hosted backend with MongoDB Atlas  
- ⚡ Fast deployment on Netlify + Render  

---

## 🛠️ Tech Stack
- **Frontend:** React, Axios, CSS  
- **Backend:** Node.js, Express, Mongoose  
- **Database:** MongoDB Atlas  
- **Deployment:** Netlify (frontend), Render (backend)  

---

## 📂 Project Structure
- frontend/   → React app (Netlify)
- backend/    → Node.js + Express API (Render)
